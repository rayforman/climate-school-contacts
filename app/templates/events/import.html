Instructions

Upload a CSV file containing attendee names. The system will:

  1. Look for the "First Name" and "Last Name" columns in your CSV
  2. Match these names against guests in your database
  3. Add all matched guests to this event's attendee list

Names that don't match any records in your database will be reported but not added.

CSV Format Example

Your CSV file should have these required columns:

First Name,Last Name
John,Doe
Jane,Smith
Michael,Johnson

Other columns in the CSV will be ignored.
